Existing Functionality

Original Thermostat Project

The Thermostat Project from CS 350: Emerging Systems Architectures and Technologies focused on embedded systems concepts and the interaction between software logic and hardware-oriented behavior. The original version represented thermostat functionality such as reading temperature-related input, comparing current conditions against a target value, and responding with output that simulates or controls heating behavior.

As an artifact, this project is useful because it includes practical control logic, state-based decisions, and opportunities to improve structure, reliability, and data handling. Those qualities make it a strong foundation for demonstrating growth across the three capstone enhancement categories.

Code Analysis

Target Areas for Improvement

Structure and Maintainability

The original project can be strengthened through clearer organization, more descriptive naming, and separation of responsibilities between input handling, thermostat logic, and output behavior.

Logic and Efficiency

The decision-making logic can be refined so thermostat state changes are easier to follow, test, and extend while reducing repeated or unclear conditional behavior.

Testing and Documentation

Additional comments, documentation, and test-oriented thinking can make the project easier to evaluate and improve in future iterations.

Data Handling

The project can better demonstrate database concepts by capturing thermostat events, configuration, or readings in a persistent data format.

Enhancement Plan

Planned Capstone Improvements

The enhancement plan builds on the original thermostat behavior by improving the design of the application, strengthening the logic used to evaluate thermostat state, and adding a database-oriented component for storing or analyzing relevant thermostat data.

These improvements are intended to show growth from the original course project into a more complete artifact that demonstrates professional software engineering judgment, algorithmic reasoning, and an understanding of persistent data.
